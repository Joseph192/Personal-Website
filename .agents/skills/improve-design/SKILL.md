# Improve Design Skill

## Goal

Improve the site visually while keeping it simple, readable, and maintainable.

## Workflow

1. Inspect the existing components and styles first.
2. Reuse existing design variables from `src/styles/variables.css`.
3. Keep layout responsive for mobile and desktop.
4. Prefer small design improvements over full rewrites.
5. Preserve content and routes unless asked.
6. Check accessibility:
   - semantic headings
   - readable contrast
   - useful alt text
   - descriptive links
7. Run the relevant build/check command if available.

## Rules

- Do not add a UI library unless explicitly requested.
- Do not rewrite all CSS for a small visual change.
- Do not remove existing content.
- Keep the personal tone of the site.

## Final Response

Include:
- Design changes made.
- Files changed.
- Checks run.
- Any visual/manual checks recommended.
