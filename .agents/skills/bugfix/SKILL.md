# Bugfix Skill

## Goal

Fix the bug with the smallest safe change.

## Workflow

1. Restate the bug.
2. Inspect the relevant page, component, content file, or style file.
3. Identify the likely cause.
4. Make the smallest focused fix.
5. Avoid unrelated redesigns or refactors.
6. Run the relevant check if available.
7. Explain the cause and fix.

## Common Personal Site Bugs

- Broken image paths.
- Broken internal links.
- Missing frontmatter.
- Invalid markdown.
- CSS layout overflow on mobile.
- Components receiving missing data.
- Build errors from bad imports.

## Final Response

Include:
- Root cause.
- Fix made.
- Files changed.
- Checks run.
